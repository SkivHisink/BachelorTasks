# OPK
В данной папке находятся решения к задачам курса ОПК, выполненные на 2 курсе ФФ НГУ.
## Пояснение по некоторым задачам
Ранее были введены некоторые сокращения:
* `PE` - pointer edition
* `CE` - console edition
* `NT` - don't have fool test
* `F` - foundation
* `R` - release
* `MT` - memory test
